# Generador de Números Aleatorios

## Descripción
Este proyecto es una aplicación web sencilla que permite generar números aleatorios dentro de un rango definido por el usuario.  
El usuario introduce un valor mínimo y un valor máximo, y la aplicación muestra un número aleatorio comprendido entre ambos.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
